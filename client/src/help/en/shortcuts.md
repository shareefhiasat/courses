# Keyboard Shortcuts

Quick reference for all keyboard shortcuts available in the Military LMS.

## Global shortcuts

| Shortcut | Action | Notes |
| --- | --- | --- |
| `Ctrl+K` / `Cmd+K` | Open command palette | Search and jump to any screen or help article |
| `Ctrl+Shift+R` / `Cmd+Shift+R` | Hard refresh | Bypasses browser cache; reloads all data |
| `Ctrl+F` / `Cmd+F` | In-page search | Searches within the current page content |
| `Esc` | Close dialog / overlay | Closes modals, command palette, and pop-overs |

## Navigation

| Shortcut | Action | Notes |
| --- | --- | --- |
| `Alt+H` | Go to [Home](/en/home) | Available from any authenticated screen |
| `Alt+D` | Go to [Dashboard](/en/dashboard) | Opens the last active tab |
| `Alt+N` | Go to [Notifications](/en/notifications) | Opens the notification drawer |
| `Alt+C` | Go to [Chat](/en/chat) | Opens the chat screen |
| `Alt+P` | Go to [Profile & Settings](/en/profile) | Opens your profile settings |

## Help docs (Nextra)

| Shortcut | Action | Notes |
| --- | --- | --- |
| `Ctrl+K` / `Cmd+K` | Search docs | Full-text search across all help articles |
| `←` / `→` | Previous / Next article | Navigates through sidebar order |
| `Ctrl+Click` / `Cmd+Click` | Open link in new tab | Works on any cross-link between articles |

## Browser-level shortcuts

| Shortcut | Action | Notes |
| --- | --- | --- |
| `F11` | Toggle fullscreen | Useful for the QR scanner and calendar views |
| `Ctrl++` / `Cmd++` | Zoom in | Enlarge UI for better readability |
| `Ctrl+-` / `Cmd+-` | Zoom out | Reduce UI size |
| `Ctrl+0` / `Cmd+0` | Reset zoom | Restore default zoom level |

## Tips

- **Mac users** — Replace `Ctrl` with `Cmd` (⌘) for all shortcuts except `Ctrl+K` which works with either.
- **Keyboard-only navigation** — Use `Tab` to move between interactive elements and `Enter` to activate them. The LMS supports full keyboard navigation.
- **Screen reader compatibility** — All interactive elements have ARIA labels for screen reader users.
