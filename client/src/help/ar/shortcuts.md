# اختصارات لوحة المفاتيح

مرجع سريع لجميع اختصارات لوحة المفاتيح المتاحة في نظام إدارة التعلم العسكري.

## اختصارات عامة

| الاختصار | الإجراء | ملاحظات |
| --- | --- | --- |
| `Ctrl+K` / `Cmd+K` | فتح لوحة الأوامر | ابحث واقفز إلى أي شاشة أو مقال مساعدة |
| `Ctrl+Shift+R` / `Cmd+Shift+R` | تحديث قسري | يتجاوز ذاكرة التخزين المؤقت للمتصفح؛ يعيد تحميل جميع البيانات |
| `Ctrl+F` / `Cmd+F` | البحث في الصفحة | يبحث ضمن محتوى الصفحة الحالية |
| `Esc` | إغلاق مربع / طبقة | يغلق النوافذ المنبثقة ولوحة الأوامر والقوائم |

## التنقل

| الاختصار | الإجراء | ملاحظات |
| --- | --- | --- |
| `Alt+H` | الذهاب إلى [الرئيسية](/ar/home) | متاح من أي شاشة مصادق عليها |
| `Alt+D` | الذهاب إلى [لوحة التحكم](/ar/dashboard) | يفتح آخر تبويب نشط |
| `Alt+N` | الذهاب إلى [الإشعارات](/ar/notifications) | يفتح درج الإشعارات |
| `Alt+C` | الذهاب إلى [الدردشة](/ar/chat) | يفتح شاشة الدردشة |
| `Alt+P` | الذهاب إلى [الملف الشخصي والإعدادات](/ar/profile) | يفتح إعدادات ملفك الشخصي |

## وثائق المساعدة (Nextra)

| الاختصار | الإجراء | ملاحظات |
| --- | --- | --- |
| `Ctrl+K` / `Cmd+K` | بحث في الوثائق | بحث نصي كامل عبر جميع مقالات المساعدة |
| `←` / `→` | المقال السابق / التالي | يتنقل عبر ترتيب الشريط الجانبي |
| `Ctrl+Click` / `Cmd+Click` | فتح الرابط في تبويب جديد | يعمل على أي رابط متبادل بين المقالات |

## اختصارات على مستوى المتصفح

| الاختصار | الإجراء | ملاحظات |
| --- | --- | --- |
| `F11` | تبديل ملء الشاشة | مفيد لماسح QR وعرض التقويم |
| `Ctrl++` / `Cmd++` | تكبير | تكبير الواجهة لقراءة أفضل |
| `Ctrl+-` / `Cmd+-` | تصغير | تصغير حجم الواجهة |
| `Ctrl+0` / `Cmd+0` | إعادة ضبط التكبير | استعادة مستوى التكبير الافتراضي |

## نصائح

- **مستخدمو Mac** — استبدل `Ctrl` بـ `Cmd` (⌘) لجميع الاختصارات باستثناء `Ctrl+K` الذي يعمل بكليهما.
- **التنقل بلوحة المفاتيح فقط** — استخدم `Tab` للتنقل بين العناصر التفاعلية و `Enter` لتفعيلها. النظام يدعم التنقل الكامل بلوحة المفاتيح.
- **توافق قارئ الشاشة** — جميع العناصر التفاعلية لها تسميات ARIA لمستخدمي قارئ الشاشة.
