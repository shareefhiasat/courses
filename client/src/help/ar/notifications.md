# الإشعارات

شاشة الإشعارات تعرض جميع إشعارات النظام في مكان واحد — الإعلانات، نتائج الاختبارات، تنبيهات الحضور، مهام سير العمل، والمزيد.

## من يمكنه الوصول

| الدور | العمليات | ما يمكنهم فعله |
| --- | --- | --- |
| المشرف العام | عرض، تحديث | عرض وإدارة جميع الإشعارات |
| المشرف | عرض، تحديث | عرض وإدارة الإشعارات |
| الموارد البشرية | عرض، تحديث | عرض وإدارة الإشعارات |
| المدرب | عرض، تحديث | عرض وإدارة إشعاراتهم |
| الطالب | عرض، تحديث | عرض وإدارة إشعاراتهم |

> **معرف الشاشة:** `notifications` — يتطلب عملية `view`. `update` مطلوبة للتعليم كمقروء/غير مقروء.

## الإجراءات الرئيسية

- **قراءة الإشعارات** — انقر على إشعار لعرض تفاصيله الكاملة. بعض الإشعارات تتضمن روابط مباشرة للشاشة ذات الصلة.
- **التعليم كمقروء/غير مقروء** — تبديل حالة القراءة فردياً أو بالجملة.
- **تصفية** — تصفية حسب النوع أو نطاق التاريخ أو مستوى الأهمية.
- **الإعدادات** — تكوين أي الإشعارات تتلقاها عبر صفحة [الملف الشخصي والإعدادات](/ar/profile).

## القواعد والتحقق

- **شارة غير مقروء** — يعرض الشريط العلوي عدداً للإشعارات غير المقروءة. يتحدث في الوقت الفعلي.
- **التعليم التلقائي كمقروء** — النقر على إشعار يعلمه تلقائياً كمقروء.
- **الاحتفاظ** — تُحتفظ الإشعارات لمدة 90 يوماً. الإشعارات الأقدم تُؤرشف ولم تعد مرئية.
- **نطاق لكل مستخدم** — يمكنك فقط رؤية الإشعارات الموجهة إليك أو المبثوثة لدورك.
- **إجراءات بالجملة** — يمكنك تعليم حتى 50 إشعار كمقروء/غير مقروء في إجراء واحد.

## القيود

- لا يمكنك حذف الإشعارات — فقط تعليمها كمقروء أو غير مقروء.
- تسليم البريد الإلكتروني للإشعارات يعتمد على إعدادات ملفك الشخصي وتكوين قالب البريد.
- إشعارات الدفع (المتصفح) تتطلب منح الإذن في إعدادات المتصفح.

- [الدردشة](/ar/chat) — رسائل الدردشة تولد أيضاً إشعارات.
