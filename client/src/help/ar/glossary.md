# مسرد المصطلحات

المصطلحات الرئيسية المستخدمة في هذه الوثائق ونظام إدارة التعلم العسكري.

## A–F

| المصطلح | التعريف |
| --- | --- |
| **نافذة الحضور** | الفترة القابلة للتحرير التي يمكن خلالها تعديل سجلات الحضور. خارج هذه النافذة، السجلات مقفلة. |
| **إجراء بالجملة** | عملية تُطبق على عدة عناصر في وقت واحد (مثل تعليم 50 إشعار كمقروء). |
| **جلسة فصل** | حالة مجدولة من فصل في تقويم [الجدولة](/ar/scheduling). يُسجل الحضور لكل جلسة. |
| **لوحة الأوامر** | طبقة التنقل السريع التي تُفتح بـ `Ctrl+K` (أو `Cmd+K` على Mac). تسمح بالبحث والقفز إلى أي شاشة. |
| **تقويم الامتثال** | عرض تقويمي في [سير العمل](/ar/workflow) يعرض تواريخ الاستحقاق والعناصر المتأخرة لعمليات الموافقة. |
| **رابط متبادل** | ارتباط تشعبي في مقال يشير إلى مقال ذي صلة، مما يتيح التنقل من البداية إلى النهاية في سير العمل. |
| **ربط مباشر** | رابط URL يشير مباشرة إلى تبويب أو قسم محدد (مثل `/dashboard#programs`). |
| **تاريخ الاستحقاق** | الموعد النهائي لاختبار أو مرحلة سير عمل أو واجب. تواريخ الاستحقاق الماضية تمنع التقديمات الجديدة. |

## G–M

| المصطلح | التعريف |
| --- | --- |
| **حضور الموارد البشرية** | تسجيل الحضور بالجملة من قبل موظفي الموارد البشرية، عادةً لأحداث على مستوى المنظمة أو تصحيحات. |
| **Keycloak** | نظام إدارة الهوية والوصول الذي يتولى مصادقة المستخدمين وتسجيل الدخول وإدارة الجلسات للنظام. |
| **التحميل الكسول** | تقنية يُحمل فيها محتوى التبويب فقط عند فتح التبويب لأول مرة، مما يحسن سرعة تحميل الصفحة الأولية. |
| **التخزين المحلي** | تخزين في المتصفح يُستخدم للاحتفاظ بتفضيلات المستخدم (اللغة، السمة، التبويب النشط) عبر الجلسات. |
| **MinIO** | واجهة التخزين الكائني التي تشغل [القرص الذكي](/ar/smart-drive). تخزن جميع الملفات المرفوعة والمرفقات. |

## N–S

| المصطلح | التعريف |
| --- | --- |
| **الاحتفاظ بالإشعارات** | نافذة الـ 90 يوماً التي تكون فيها الإشعارات مرئية. الإشعارات الأقدم تُؤرشف. |
| **مصفوفة الصلاحيات** | شبكة في [الملف الشخصي والإعدادات](/ar/profile) (للمشرف العام فقط) تعرض أي الأدوار لديها أي عمليات على أي شاشات. |
| **رمز QR** | رمز فريد قابل للمسح يُولّد لكل طالب لتسجيل حضور سريع عبر ماسح QR. |
| **RBAC** | تحكم في الوصول قائم على الدور — النموذج الذي تحدد فيه أدوار المستخدمين (مشرف عام، مشرف، موارد بشرية، مدرب، طالب) وصول الشاشة والعمليات المسموح بها. |
| **Redis** | مخزن بيانات في الذاكرة يُستخدم للتخزين المؤقت والميزات الفورية مثل اتصالات WebSocket للدردشة. |
| **معرف الشاشة** | معرف فريد لكل شاشة (مثل `attendance`، `quizzes`، `drive`) يُستخدم من قبل مصفوفة الصلاحيات وسجل التنقل. |
| **الحذف الناعم** | حذف يضع علامة على السجل كمحذوف دون إزالته من قاعدة البيانات. يمكن استعادة العناصر المحذوفة ناعماً (مثل ملفات القرص الذكي في المهملات لمدة 30 يوماً). |

## T–Z

| المصطلح | التعريف |
| --- | --- |
| **استكشاف الأخطاء** | قسم في كل مقال يسرد المشاكل الشائعة وحلولها. |
| **UTC** | التوقيت العالمي المنسق — جميع الأوقات تُخزن بـ UTC وتُعرض بالمنطقة الزمنية المحلية للمستخدم. |
| **WebSocket** | اتصال ثنائي الاتجاه مستمر يُستخدم للميزات الفورية مثل تسليم رسائل [الدردشة](/ar/chat) وتحديثات الإشعارات. |
| **قالب سير العمل** | نموذج محدد مسبقاً وسلسلة موافقات لنوع طلب محدد (إجازة، تدريب، معدات، إلخ) في نظام [سير العمل](/ar/workflow). |
