# سير العمل

نظام سير العمل يدير عمليات الموافقة وتوجيه المستندات. يضمن أن الطلبات (إجازة، تدريب، معدات، إلخ) تتبع سلسلة الموافقة الصحيحة قبل تنفيذها.

## من يمكنه الوصول

| الدور | العمليات | ما يمكنهم فعله |
| --- | --- | --- |
| المشرف العام | عرض، إنشاء، تحديث، حذف | إدارة كاملة لسير العمل |
| المشرف | عرض، إنشاء، تحديث، حذف | إنشاء وإدارة مستندات سير العمل |
| الموارد البشرية | عرض، إنشاء، تحديث | بدء والموافقة على مستندات سير العمل |
| المدرب | عرض، إنشاء، تحديث | تقديم وتتبع طلبات سير العمل |
| الطالب | عرض | عرض طلباتهم المُقدمة فقط |

> **معرف الشاشة:** `workflow` — يتطلب عملية `view`. الإنشاء/التحديث/الحذف تتطلب الصلاحيات المقابلة.

## الشاشات

| الشاشة | الوصف |
| --- | --- |
| **صندوق الوارد** | عرض المهام المعينة لك التي تحتاج إجراء. تعرض العناصر المعلقة والمكتملة والمتأخرة. |
| **تفاصيل المستند** | مراجعة البيانات الوصفية لمستند سير العمل وسلسلة الموافقات والتعليقات والمرفقات. |
| **تقويم الامتثال** | عرض تواريخ الاستحقاق والعناصر المتأخرة على تقويم. يساعد في تتبع المواعيد بصرياً. |
| **التحليلات** | مؤشرات الأداء الرئيسية والرسوم البيانية لأداء سير العمل — أوقات الموافقة ومراحل الاختناق ومعدلات الإكمال. |

## الإجراءات الرئيسية

- **تقديم طلب** — إنشاء مستند سير عمل جديد باختيار قالب وملء الحقول المطلوبة.
- **موافقة/رفض** — اتخذ إجراءً على مهمة في صندوق الوارد الخاص بك. يجب تقديم تعليق عند الرفض.
- **تعليق** — أضف تعليقاً على مستند سير العمل في أي مرحلة. التعليقات مرئية لجميع المشاركين.
- **تفويض** — تعيين مهمة لمستخدم آخر. يُعلم المُعين الأصلي ويُسجل التفويض.
- **استرجاع** — يمكن لمقدم الطلب استرجاع مستند قبل إجراء أول موافقة.

## القواعد والتحقق

- **سلسلة الموافقة** — كل قالب سير عمل يحدد سلسلة موافقات مرتبة. ينتقل المستند إلى المعتمد التالي فقط بعد موافقة الحالي.
- **التعليق مطلوب عند الرفض** — لا يمكنك رفض مستند سير عمل دون تقديم سبب.
- **سجل التفويض** — جميع التفويضات تُسجل مع المُفوض والمُفوض إليه والطابع الزمني والسبب.
- **تواريخ الاستحقاق** — كل مرحلة لها تاريخ استحقاق اختياري. العناصر المتأخرة تظهر بالأحمر في صندوق الوارد وتقويم الامتثال.
- **قفل المستند** — بينما يُراجع مستند، يكون مقفلاً — لا يمكن لمقدم الطلب تحريره حتى تكتمل المراجعة أو يُسترجع المستند.
- **التصعيد التلقائي** — إذا لم يتخذ المعتمد إجراءً ضمن المهلة المُكونة، يمكن تصعيد المهمة تلقائياً إلى مشرفه.

## القيود

- لا يمكنك تحرير مستند سير العمل بعد أول موافقة — يجب استرجاعه أولاً.
- الموافقات المتوازية (عدة معتمدين في نفس المرحلة) غير مدعومة في الإصدار الحالي.
- بيانات التحليلات تُجمع يومياً — مقاييس سير العمل الفورية غير متاحة.
