# القرص الذكي

القرص الذكي هو نظام إدارة الملفات لتخزين وتنظيم ومشاركة مواد التعلم. مدعوم بتخزين MinIO الكائني.

## من يمكنه الوصول

| الدور | العمليات | ما يمكنهم فعله |
| --- | --- | --- |
| المشرف العام | عرض، إنشاء، تحديث، حذف | إدارة كاملة للقرص |
| المشرف | عرض، إنشاء، تحديث، حذف | إدارة كاملة للقرص |
| الموارد البشرية | عرض، إنشاء، تحديث، حذف | رفع وإدارة الملفات |
| المدرب | عرض، إنشاء، تحديث، حذف | رفع ومشاركة الموارد مع الطلاب |
| الطالب | عرض، إنشاء | رفع الملفات وعرض الموارد المشتركة |

> **معرف الشاشة:** `drive` — يتطلب عملية `view`. الإنشاء/التحديث/الحذف تتطلب الصلاحيات المقابلة.

## الإجراءات الرئيسية

- **رفع الملفات** — اسحب وأفلت أو تصفح لرفع الملفات والمجلدات. الصيغ المدعومة تشمل المستندات والصور والفيديو و PDF.
- **معاينة** — انقر على ملف لمعاينته في المتصفح. الصيغ المدعومة: PDF، الصور، النص، وبعض صيغ الفيديو.
- **تنزيل** — تنزيل الملفات فردياً أو اختر عدة ملفات للتنزيل كأرشيف ZIP.
- **مشاركة** — تعيين صلاحيات لمشاركة الملفات مع مستخدمين أو مجموعات أو فصول كاملة محددة.
- **تنظيم** — إنشاء مجلدات ومجلدات فرعية لتنظيم ملفاتك هرمياً.
- **المهملات** — الملفات المحذوفة تذهب إلى مجلد المهملات ويمكن استعادتها خلال 30 يوماً. بعد 30 يوماً، تُحذف الملفات نهائياً.
- **بحث** — استخدم شريط البحث للعثور على الملفات بالاسم أو البيانات الوصفية.

## القواعد والتحقق

- **حد حجم الملف** — الملفات الفردية محدودة بـ 500 ميجابايت. للملفات الأكبر، اتصل بمدير النظام.
- **حصة التخزين** — كل مستخدم لديه حصة تخزين. عند الوصول للحد، يُحظر الرفع حتى تُحذف الملفات.
- **أنواع الملفات المسموحة** — الملفات التنفيذية (`.exe`، `.bat`، `.sh`) محظورة للأمان. أنواع الملفات الأخرى مسموحة.
- **صلاحيات المشاركة** — يمكنك تعيين صلاحيات عرض فقط أو عرض وتنزيل عند مشاركة الملفات.
- **الحذف الناعم** — جميع الحذفات هي حذفات ناعمة. تبقى الملفات في المهملات لمدة 30 يوماً قبل الحذف الدائم.
- **سجل الإصدارات** — رفع ملف بنفس اسم ملف موجود ينشئ إصداراً جديداً. الإصدارات السابقة متاحة عبر لوحة تفاصيل الملف.

## القيود

- عمق المجلد محدود بـ 10 مستويات.
- التنزيل بالجملة (ZIP) محدود بـ 2 جيجابايت إجمالي الحجم.
- معاينات الملفات غير متاحة لجميع صيغ الفيديو — بعضها قد يتطلب التنزيل أولاً.
- الملفات المشتركة لا تُزامن في الوقت الفعلي — يحتاج المستلمون إلى تحديث عرض القرص لرؤية العناصر المشتركة حديثاً.
