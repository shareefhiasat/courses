# Documentation: E2E Testing and Playwright Scaffold

- This folder contains guidance and code scaffolds for end-to-end testing.
- Use Ctrl+K to search for topics like "Sign-in tests" or "Playwright configuration".
- Run tests locally with environment variables for credentials.

Key files:
- tests/e2e/login.spec.js
- tests/e2e/playwright.config.js
- docs/e2e-guide.md

How to contribute:
- Add more tests under tests/e2e/ with clear naming.
- Update docs/e2e-guide.md with new practices and scenarios.
