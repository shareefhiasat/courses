# Internationalization (i18n) Module

## Business Context
The Military LMS supports Arabic and English. All UI text, API responses, and data models support both languages. The i18n module manages translations and language switching.

## UI Pages
- Language toggle in header (AR/EN)

## Business Rules
- Supported languages: Arabic (ar), English (en)
- RTL layout for Arabic
- All database entities have nameEn and nameAr fields
- API responses include both language fields
- Language preference stored per user
- Fallback to English if Arabic translation missing

## Test Coverage
- Currently no dedicated i18n spec file
- i18n coverage embedded in other module tests (Arabic field assertions)
